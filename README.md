# Cursor Clone

A simple landing page clone of [Cursor](https://cursor.com/) built using HTML and CSS.

Preview: ![Cursor Clone Preview](https://ik.imagekit.io/zk9yllrd0/cursor-clone-beryl.vercel.app.png)

## Features

- Navigation bar
- Hero section with download button
- Responsive layout
- Asset integration

## Technologies Used

- HTML5
- CSS3

## Getting Started

To view the project, simply open the `index.html` file in any modern web browser.
